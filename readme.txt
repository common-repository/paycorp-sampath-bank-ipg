== Description ==
Paycorp Payment Gateway is the only Visa, MasterCard accredited and certified payment solutions provider for Sampath Bank.  With the request of the Banks, Paycorp have created a secure plugin that directly talks to the Paycorp Secure Systems without using any third parties or intermediaries.

This plugin is the only certified PCI DSS plugin in Sri lanka that is built and managed by Paycorp a level 1 PCI DSS Compliant Gateway. Please ensure you use this plugin to avoid any breach of data compliance and risk your customers card data being compromised.

WHY Choose our plugin than the others?
1. Our plugin is the only secure plugin that is hosted and managed internally by Paycorp.
2. Why go through third parties to process payments when you can come direct to the supplier
3. Easy integration to accept the Credit and Debit card payments through Sampath Bank
4. No Technical Skills needed.
5. You can fully customise the payment page with CSS or add logos etc
6. Accept payments from any currency, international and local and international debit and credit card acceptance
7. Secure console login to view transactions that allows you to void, refund transactions
8. Peace of Mind that your customers data is fully protected by the  NO 1 Gateway in Sri Lanka.

== Installation ==
1. Install WordPress.
2. Ensure you have the latest version of WooCommerce plugin installed
3. Upload plugin zip file via wordpress admin panel or Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
4. Activate the plugin through the ‘Plugins’ menu available in WordPress
5. Enter payment gateway details in woocommerce->settings->checkout->paycorp_sampath_ipg
6. Make sure to Enable Sampath IPG Module.
